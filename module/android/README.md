# [Android App] Project Forge Module

This directory contains the files used by the "android" module of [Project Forge](https://projectforge.dev).

## Purpose

Webview-based application and Android build

## Usage

When the "android" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/android.md](doc/module/android.md) for usage information.
