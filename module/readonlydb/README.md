# [Read-only DB] Project Forge Module

This directory contains the files used by the "readonlydb" module of [Project Forge](https://projectforge.dev).

## Purpose

Adds a read-only database connection, with a separate configuration

## Usage

When the "readonlydb" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/readonlydb.md](doc/module/readonlydb.md) for usage information.
