# [Brand Icons] Project Forge Module

This directory contains the files used by the "brands" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides thousands of SVG icons from [simple-icons] for representing common logos

## Usage

When the "brands" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/brands.md](doc/module/brands.md) for usage information.
