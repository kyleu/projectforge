# [Theme Catalog] Project Forge Module

This directory contains the files used by the "themecatalog" module of [Project Forge](https://projectforge.dev).

## Purpose

Includes a dozen default themes, and facilities to create additional

## Usage

When the "themecatalog" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/themecatalog.md](doc/module/themecatalog.md) for usage information.
