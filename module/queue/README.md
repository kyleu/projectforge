# [Queue] Project Forge Module

This directory contains the files used by the "queue" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a simple message queue based on SQLite

## Usage

When the "queue" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/queue.md](doc/module/queue.md) for usage information.
