# [Audit] Project Forge Module

This directory contains the files used by the "audit" module of [Project Forge](https://projectforge.dev).

## Purpose

Using the database module, provides an audit framework for tracking changes

## Usage

When the "audit" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/audit.md](doc/module/audit.md) for usage information.
