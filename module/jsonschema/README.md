# [JSON Schema] Project Forge Module

This directory contains the files used by the "jsonschema" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a representation of JSON Schema files and utilities for working with them

## Usage

When the "jsonschema" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/jsonschema.md](doc/module/jsonschema.md) for usage information.
