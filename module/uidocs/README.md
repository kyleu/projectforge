# [Metamodel] Project Forge Module

This directory contains the files used by the "metamodel" module of [Project Forge](https://projectforge.dev).

## Purpose

Golang representation of models, enums, and types

## Usage

When the "metamodel" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/metamodel.md](doc/module/metamodel.md) for usage information.
