# [SQL Server] Project Forge Module

This directory contains the files used by the "sqlserver" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for accessing MSSQL databases

## Usage

When the "sqlserver" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/sqlserver.md](doc/module/sqlserver.md) for usage information.
