# [Desktop] Project Forge Module

This directory contains the files used by the "desktop" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a desktop application using the system's webview

## Usage

When the "desktop" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/desktop.md](doc/module/desktop.md) for usage information.
