# [WASM Server] Project Forge Module

This directory contains the files used by the "wasmserver" module of [Project Forge](https://projectforge.dev).

## Purpose

Build your normal app as an HTTP server, but load it as a WebAssembly module or Service Worker in the browser.

## Usage

When the "wasmserver" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/wasmserver.md](doc/module/wasmserver.md) for usage information.
