# [Upgrade] Project Forge Module

This directory contains the files used by the "upgrade" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides in-place version upgrades using GitHub Releases

## Usage

When the "upgrade" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/upgrade.md](doc/module/upgrade.md) for usage information.
