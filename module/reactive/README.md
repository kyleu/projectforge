# [Reactive Values] Project Forge Module

This directory contains the files used by the "reactive" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides thread-safe reactive values with observer pattern support

## Usage

When the "reactive" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/reactive.md](doc/module/reactive.md) for usage information.
