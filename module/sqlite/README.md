# [SQLite] Project Forge Module

This directory contains the files used by the "sqlite" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for accessing SQLite databases

## Usage

When the "sqlite" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/sqlite.md](doc/module/sqlite.md) for usage information.
