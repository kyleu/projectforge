# [PostgreSQL] Project Forge Module

This directory contains the files used by the "postgres" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for accessing PostgreSQL databases

## Usage

When the "postgres" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/postgres.md](doc/module/postgres.md) for usage information.
