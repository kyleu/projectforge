# [User] Project Forge Module

This directory contains the files used by the "user" module of [Project Forge](https://projectforge.dev).

## Purpose

Classes for representing a user

## Usage

When the "user" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/user.md](doc/module/user.md) for usage information.
