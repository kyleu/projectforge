# [Numerics] Project Forge Module

This directory contains the files used by the "numeric" module of [Project Forge](https://projectforge.dev).

## Purpose

TypeScript and Golang implementations for managing large numbers.

## Usage

When the "numeric" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/numeric.md](doc/module/numeric.md) for usage information.
