# [WASM Client] Project Forge Module

This directory contains the files used by the "wasmclient" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a WebAssembly library and HTML host for an HTTP client

## Usage

When the "wasmclient" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/wasmclient.md](doc/module/wasmclient.md) for usage information.
