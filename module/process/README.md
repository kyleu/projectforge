# [Process] Project Forge Module

This directory contains the files used by the "process" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a framework for managing system processes

## Usage

When the "process" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/process.md](doc/module/process.md) for usage information.
