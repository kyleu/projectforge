# [Grep Search] Project Forge Module

This directory contains the files used by the "grep" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a mechanism for searching with ripgrep

## Usage

When the "grep" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/grep.md](doc/module/grep.md) for usage information.
