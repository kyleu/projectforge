# [Marketing Site] Project Forge Module

This directory contains the files used by the "marketing" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a website for downloads, tutorials, and marketing

## Usage

When the "marketing" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/marketing.md](doc/module/marketing.md) for usage information.
