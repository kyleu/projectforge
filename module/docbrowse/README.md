# [Doc Browser] Project Forge Module

This directory contains the files used by the "docbrowse" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a UI for browsing the documentation

## Usage

When the "docbrowse" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/docbrowse.md](doc/module/docbrowse.md) for usage information.
