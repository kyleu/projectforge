# [Types] Project Forge Module

This directory contains the files used by the "types" module of [Project Forge](https://projectforge.dev).

## Purpose

Classes for representing common data types, along with helpers for converting and rendering them.

## Usage

When the "types" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/types.md](doc/module/types.md) for usage information.
