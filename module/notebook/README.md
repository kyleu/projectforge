# [Notebook] Project Forge Module

This directory contains the files used by the "notebook" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an Observable Framework notebook

## Usage

When the "notebook" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/notebook.md](doc/module/notebook.md) for usage information.
