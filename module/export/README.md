# [Export] Project Forge Module

This directory contains the files used by the "export" module of [Project Forge](https://projectforge.dev).

## Purpose

Generates code based on the project's schema

## Usage

When the "export" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/export.md](doc/module/export.md) for usage information.
