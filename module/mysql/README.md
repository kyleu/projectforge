# [MySQL] Project Forge Module

This directory contains the files used by the "mysql" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for accessing MySQL databases

## Usage

When the "mysql" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/mysql.md](doc/module/mysql.md) for usage information.
