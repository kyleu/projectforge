# [Task Engine] Project Forge Module

This directory contains the files used by the "task" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an engine for executing and monitoring tasks

## Usage

When the "task" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/task.md](doc/module/task.md) for usage information.
