# [MCP Server] Project Forge Module

This directory contains the files used by the "mcp" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a CLI interface for running Model Context Protocol requests

## Usage

When the "mcp" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/mcp.md](doc/module/mcp.md) for usage information.
