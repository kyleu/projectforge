# [Expression] Project Forge Module

This directory contains the files used by the "expression" module of [Project Forge](https://projectforge.dev).

## Purpose

Exposes Common Expression Language (CEL) engine for evaluating arbitrary expressions

## Usage

When the "expression" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/expression.md](doc/module/expression.md) for usage information.
