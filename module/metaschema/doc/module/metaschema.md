# Metaschema

The **`metaschema`** module provides transformations between **`metamodel`** and **`jsonschema`** representations.

## Dependencies

- **Required**: [`jsonschema`](jsonschema.md) - JSON Schema representation
- **Required**: [`metamodel`](metamodel.md) - Internal enum/model types
