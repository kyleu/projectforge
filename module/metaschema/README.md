# [Metaschema] Project Forge Module

This directory contains the files used by the "metaschema" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides transformations between metamodel and jsonschema representations

## Usage

When the "metaschema" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/metaschema.md](doc/module/metaschema.md) for usage information.
