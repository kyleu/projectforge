# [Scheduled Jobs] Project Forge Module

This directory contains the files used by the "schedule" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a scheduled job engine and UI based on gocron

## Usage

When the "schedule" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/schedule.md](doc/module/schedule.md) for usage information.
