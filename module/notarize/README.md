# [Notarize] Project Forge Module

This directory contains the files used by the "notarize" module of [Project Forge](https://projectforge.dev).

## Purpose

Sends files to Apple for notarization

## Usage

When the "notarize" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/notarize.md](doc/module/notarize.md) for usage information.
