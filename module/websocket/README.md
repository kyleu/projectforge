# [WebSocket] Project Forge Module

This directory contains the files used by the "websocket" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for hosting WebSocket connections

## Usage

When the "websocket" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/websocket.md](doc/module/websocket.md) for usage information.
