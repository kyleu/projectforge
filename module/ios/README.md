# [iOS App] Project Forge Module

This directory contains the files used by the "ios" module of [Project Forge](https://projectforge.dev).

## Purpose

Webview-based application and iOS build

## Usage

When the "ios" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/ios.md](doc/module/ios.md) for usage information.
