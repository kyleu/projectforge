# [Search] Project Forge Module

This directory contains the files used by the "search" module of [Project Forge](https://projectforge.dev).

## Purpose

Adds search facilities to the top-level navigation bar

## Usage

When the "search" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/search.md](doc/module/search.md) for usage information.
