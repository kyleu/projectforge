# [HTTP Archive] Project Forge Module

This directory contains the files used by the "har" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides classes for parsing HTTP Archive (*.har) files

## Usage

When the "har" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/har.md](doc/module/har.md) for usage information.
