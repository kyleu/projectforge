# [Sandbox] Project Forge Module

This directory contains the files used by the "sandbox" module of [Project Forge](https://projectforge.dev).

## Purpose

Useful playgrounds for testing custom functions

## Usage

When the "sandbox" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/sandbox.md](doc/module/sandbox.md) for usage information.
