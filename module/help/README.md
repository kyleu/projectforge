# [Help] Project Forge Module

This directory contains the files used by the "help" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides Markdown help files that integrate into the UI

## Usage

When the "help" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/help.md](doc/module/help.md) for usage information.
