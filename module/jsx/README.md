# [JSX] Project Forge Module

This directory contains the files used by the "jsx" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides a slim JSX implementation for scripting

## Usage

When the "jsx" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/jsx.md](doc/module/jsx.md) for usage information.
