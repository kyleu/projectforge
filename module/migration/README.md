# [Migration] Project Forge Module

This directory contains the files used by the "migration" module of [Project Forge](https://projectforge.dev).

## Purpose

Database migrations and a common database pool

## Usage

When the "migration" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/migration.md](doc/module/migration.md) for usage information.
