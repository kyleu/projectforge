# [GraphQL] Project Forge Module

This directory contains the files used by the "graphql" module of [Project Forge](https://projectforge.dev).

## Purpose

Supports GraphQL APIs within your application

## Usage

When the "graphql" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/graphql.md](doc/module/graphql.md) for usage information.
