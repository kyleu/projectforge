# [OpenAPI] Project Forge Module

This directory contains the files used by the "openapi" module of [Project Forge](https://projectforge.dev).

## Purpose

Embeds the Swagger UI, using your OpenAPI specification

## Usage

When the "openapi" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/openapi.md](doc/module/openapi.md) for usage information.
