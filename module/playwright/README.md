# [Playwright] Project Forge Module

This directory contains the files used by the "playwright" module of [Project Forge](https://projectforge.dev).

## Purpose

Adds a project for testing the UI using playwright.dev

## Usage

When the "playwright" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/playwright.md](doc/module/playwright.md) for usage information.
