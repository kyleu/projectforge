# Demo

The **`demo`** module uses [VHS](https://github.com/charmbracelet/vhs) and other tools to provide demo content and assets for your application.

## Overview

Adds Charmbracelet VHS tapes and helper scripts for recording repeatable demos of your application.
It does not change runtime behavior; it only ships demo assets under `tools/demo` for use in docs or marketing.
