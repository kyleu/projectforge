# [Plot] Project Forge Module

This directory contains the files used by the "plot" module of [Project Forge](https://projectforge.dev).

## Purpose

Library for visualizing data using Observable Plot

## Usage

When the "plot" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/plot.md](doc/module/plot.md) for usage information.
