# [Proxy] Project Forge Module

This directory contains the files used by the "proxy" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an HTTP proxy while still enforcing your app's security

## Usage

When the "proxy" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/proxy.md](doc/module/proxy.md) for usage information.
