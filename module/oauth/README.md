# [OAuth] Project Forge Module

This directory contains the files used by the "oauth" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides logins and session management for many OAuth providers

## Usage

When the "oauth" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/oauth.md](doc/module/oauth.md) for usage information.
