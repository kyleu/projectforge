# [Scripting] Project Forge Module

This directory contains the files used by the "scripting" module of [Project Forge](https://projectforge.dev).

## Purpose

Allows the execution of JavaScript files using a built-in interpreter

## Usage

When the "scripting" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/scripting.md](doc/module/scripting.md) for usage information.
