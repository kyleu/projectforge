# [Git] Project Forge Module

This directory contains the files used by the "git" module of [Project Forge](https://projectforge.dev).

## Purpose

Helper classes for performing operations on git repositories

## Usage

When the "git" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/git.md](doc/module/git.md) for usage information.
