# [Filesystem] Project Forge Module

This directory contains the files used by the "filesystem" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an abstraction around local and remote filesystems

## Usage

When the "filesystem" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/filesystem.md](doc/module/filesystem.md) for usage information.
