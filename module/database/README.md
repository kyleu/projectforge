# [Database] Project Forge Module

This directory contains the files used by the "database" module of [Project Forge](https://projectforge.dev).

## Purpose

Provides an API for accessing relational databases

## Usage

When the "database" module is enabled in your project, all of the files in this directory (except this readme) will be processed and included in your application.

See [doc/module/database.md](doc/module/database.md) for usage information.
