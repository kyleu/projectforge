# Doctor

The Doctor runs health checks to make sure your environment can build and run Project Forge projects.

## What it checks

- Required tools (Go, Node, Git, etc.)
- Platform-specific dependencies for enabled modules
- Optional tooling for build targets (desktop/mobile/WASM)

## How to use it

- Open `/admin/doctor` in the UI and run the checks.
- Follow the remediation steps shown for any failures.
